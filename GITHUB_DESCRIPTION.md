# GitHub Repository Description (300 words)

Modern electronics store web application - fully functional e-commerce platform built with React 19, TypeScript, and TailwindCSS 4. Users can search, filter, add to cart, and place orders for products. The project features a feature-based architecture, state management with React Context API and TanStack Query, JWT authentication, responsive design, and full Azerbaijani language localization. The home page includes hero section, promotional banners, category grid, featured products, discounts, and popular products sections. The products page features search, filtering (price, brand, category), pagination, and product detail functionality. Shopping cart is persisted with LocalStorage, supports quantity changes and order summary calculations. Multi-step checkout process is validated with Formik and Yup. Users can register, login, and create wishlists. JWT token authentication and automatic refresh token mechanism ensure security. Toast notifications, loading states, error handling, and skeleton loaders enhance user experience. API integration uses Axios, supports RESTful endpoints, pagination, filtering, and search. The project uses Vite build tool, TypeScript type safety, React Router DOM routing, Framer Motion animations, and Lucide React icon library.
